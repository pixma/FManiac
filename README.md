FManiac
=======

A simple fm radio small sized gadget, runs on li-ion battery.

-------------------------------------------

### Has features
- FM radio module.
- LCD screen.
- Capacitive touch sensor button.
- Li-ion battery support.
- Battery charging.
- 4 pin audio jack/ 3.5mm audio jack audio output.

### FM radio module
- TEA5767 FM Radio Module


### MCU selection
- atmega328
